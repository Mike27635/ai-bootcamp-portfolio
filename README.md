# Week 1 Diabetes EDA Project

## Overview
This project focuses on Exploratory Data Analysis (EDA) using the Diabetes Dataset from Kaggle. The analysis was carried out using Python, Pandas, Matplotlib, and Seaborn in Google Colab.

## Dataset
Diabetes Dataset:
https://www.kaggle.com/datasets/akshaydattatraykhare/diabetes-dataset

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Data loading and exploration
- Checking missing values
- Summary statistics
- Correlation analysis
- Data visualization using charts

## Visualizations Included
- Outcome count plot
- Glucose distribution histogram
- Age distribution histogram
- BMI vs Outcome boxplot
- Correlation heatmap

## Key Insight
The analysis showed that glucose level has the strongest relationship with diabetes outcome. Patients with higher glucose levels were more likely to have diabetes.
